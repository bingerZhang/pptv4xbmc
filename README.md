# PPTV video plugin for XBMC #

A video plugin for XBMC to watch live and non-live videos on <http://www.pptv.com/>

## Introduction ##

This XBMC plugin support online videos on <http://www.pptv.com/> (include live streams)

This is a cross-platform plugin, now support different video qualities:

* low quality
* high definition
* full high definition
* blue-ray

iPad definition m3u8 format is enabled by default, if can't play, 
you can turn it off in settings interface

Live video only support iPad definition m3u8 format

You can change default video quality in PPTV video plugin's settings interface

Your XBMC system should be capable of m3u and m3u8 streams to play iPad FullHD videos

**This plugin only get video from <http://www.pptv.com/>, all copyrights are from this site**

## 介绍 ##

PPTV 视频 XBMC 插件支持观看 <http://www.pptv.com/> 各种视频（包括直播流媒体视频）

此插件为跨平台，理论上可以在任何 XBMC 标准系统中使用，现支持以下视频质量：

* 标准（流畅）
* 高清
* 超清
* 蓝光

iPad 视频 (m3u 及 m3u8 格式) 默认为启用，如果无法正常播放，
您可以在设置界面中关闭此选项

直播视频目前只支持 iPad 的 m3u8 格式

您可以在 PPTV 视频 XBMC 插件的设置界面中修改默认的视频质量

如果需要播放 iPad 超清视频，您的 XBMC 系统需要支持播放 m3u 及 m3u8 网络媒体

**此插件只是从 <http://www.pptv.com/> 网站获取视频内容, 所有视频版权来自此网站，本插件一概不负责**
